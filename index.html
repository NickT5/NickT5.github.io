RMNDR

FLASK | RASPBERRY PI

RMNDR

— Project 3

RMNDR is an application to remind me what tasks I have to do. The user can add, edit, delete and change the status of the tasks which are stored in a database. A python script sends me an email every day with all the uncompleted tasks. Authentication is also implemented. The web application runs on a Raspberry Pi.
